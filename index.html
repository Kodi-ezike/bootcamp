Basic HTML

HTML is a markup language that uses a special syntax or notation to describe the structure of a webpage to the browser. HTML elements usually have opening and closing tags that surround and give meaning to content. For example, different elements can describe text as a heading, paragraph, or list item.There are various tags used in HTML for different functions.

Basic CSS

CSS, or Cascading Style Sheets, tells the browser how to display the text and other content that you write in HTML. With CSS, you can control the color, font, size, spacing, and many other aspects of HTML elements.

CSS Properties

Color: This is used to change the color of a text.

h1{ color:red; }

Font size: This is used to increase or decrease the font size of a text. CSS has a lot of different measurements, but most things are measured in px, em, or rem.

h1{ font-size: 40px; }

Font weight: This is used to increase the font weight of a text, to make it bold or normal. The font weight of a heading tag is bold on default.

p{ font-weight: bold; }

Text-decoration: This is used to underline or strike-through a text in different styles such as dashed, dotted etc.

h1{ text-decoration: underline dashed; }

Text-tranform: This is used to change the case of a text to upper or lowercase, can also be used to capitalize the first letter of each word.

h1{ text-tranform: capitalize; }

Border: This is used to set the border around an item. It can also be given a width, height and radius to give it that curved look at the edge.

h1{

border: 1px solid red;

border-width: 50px;

}

Height: This is used to modify the height of an element, image or container. Otherwise, it takes the default height of it's content.

img{ height: 400px; }

Width: This is used to modify the width of an element, image or container. Otherwise, it takes 100% as much width as it can.

img{ width: 400px; }

Padding: This is used to create space inside an element.

div{ padding: 2rem; }

Margin: This is used to create space outside an element.

div{ margin: 2rem; }
